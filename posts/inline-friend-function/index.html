Определение тела friend функции внутри класса

Век живи - век учись. Сейчас внезапно для себя обнаружил, что в C++ тело friend-функции можно определять прямо внутри класса: 

struct cls {
    friend void foo(cls & c) {
    }
};

// ...

cls c;
foo(c);
